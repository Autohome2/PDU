CAN Messages
============

Input
-----

* `0x500` Enable Channel
* `0x501` Disable Channel
* `0x502` Disable All Channels


Output
------

* `0x510` System Status
* `0x511` Channel Status
* `0x512` Channel Currents
* `0x513` Channel Tripped


Config
------

* `0x520` System Config
* `0x521` Channel Config
* `0x522` Input Config
